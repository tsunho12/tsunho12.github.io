## Picgo配置
- 设置仓库名：`用户名/仓库名`
- 设置分支名：之前默认是`master`，现在是`main`
- 设置Token：就是刚刚复制的那一段令牌密令
- 下面两个非必填

## Jsdelivr cdn加速
默认使用github的链接有时候不稳定
`https://raw.githubsercontent.com/用户名/仓库名/分支`
下面我们该成cdn加速
`https://cdn.jsdelivr.net/gh/GitHub用户名/仓库名`

## Picgo快捷方式
- 从剪切板上传图片到图床
- Windows/Unix: `Ctrl + Alt + u`
- macOS: `Cmd + Opt + u`

![gif演示](https://cdn.jsdelivr.net/gh/tsunho12/photos_library/img/20220414204956.png)

- 从文件夹上传图片到图床
- Windows/Unix: `Ctrl + Alt + e`
- macOS: `Cmd + Opt + e`

![gif演示](https://cdn.jsdelivr.net/gh/tsunho12/photos_library/img/20220414205101.png)

- 从输入框中输入图片名称上传到图床
- Windows/Unix: `Ctrl + Alt + o`
- macOS: `Cmd + Opt + o`
